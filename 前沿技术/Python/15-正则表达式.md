# 2 匹配规则

## 2.1 普通字符

```python
import regex as re

reg_str = "1a0b0c4d4e5f6g4"
print(re.findall("a", reg_str))
```

## 2.2 元字符

```python
import regex as re

reg_str = "1a0b0c4d4e5f6g4"
print(re.findall("a\\d", reg_str))
```

## 2.3 字符集

```python
import regex as re

reg_str = "1a0b0c4d4e5f6g4"
print(re.findall("\\d[abc]", reg_str))
print(re.findall("\\d[a-z]", reg_str))
```
