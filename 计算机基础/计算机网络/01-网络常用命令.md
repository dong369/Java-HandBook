## 1. Linux环境

### 2.1 端口情况

```shell
ps -ef | grep java
netstat -nltp                    // 查看端口信息
netstat -nltp | grep LISTEN
netstat -nltp | grep java
netstat -nlp | grep java
netstat -nlp | grep :8101
lsof -i:8101                    // 获取Pid
lsof -p pid                     // 根据Pid找服务路径
kill -9 PID                     // 强制关闭指定 PID所对应的进程
jobs
```
