## 1. 概念理解

几种IO的综合对比：

|               | BIO    | NIO      | AIO    |
| ------------- | ------ | -------- | ------ |
| 客户端:线程数 | 1:1    | M:1      | M:0    |
| 阻塞类型      | 阻塞   | 非阻塞   | 非阻塞 |
| 同步类型      | 同步   | 同步     | 异步   |
| 编程难度      | 简单   | 非常复杂 | 复杂   |
| 调试难度      | 简单   | 复杂     | 复杂   |
| 可靠性        | 非常差 | 高       | 高     |
| 吞吐量        | 低     | 高       | 高     |
